# Resume Command

Detect incomplete work and offer to continue. No arguments needed.

## Steps

1. Run: `node scripts/resume.js`
2. Display the output directly.
3. Ask the instructor which item to pick up first.
4. If partial files exist, offer to continue them.
5. If failed tasks exist, offer to retry them.

## After Completion - Show Related Skills
> `/build` build something &nbsp;|&nbsp; `/batch [class] [module]` batch generate &nbsp;|&nbsp; `/status [class]` check progress &nbsp;|&nbsp; `/daily-report` full summary
